# Cam2USB
Utilizing the CYUSB3065-BZXC chip from Infineon, which is a CX3 MIPI CSI-2 to USB bridge controller, to make my own webcam. I wanted a plug-and-play option for my webcam, but a lot of tutorials online use a NVIDIA Jetson or Raspberry Pi.
